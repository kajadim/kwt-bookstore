
# Bookstore

This is an interactive web application that simulates an online bookstore experience. The project focuses on creating a seamless user interface for browsing books, filtering by categories, and managing a shopping cart in real-time.

## Technologies Used
* HTML – Semantic structure.

* CSS – Custom styling, animations, and layouts.

* JavaScript – DOM manipulation, cart logic, and data handling.

### How to Run the Project
Option 1: Visual Studio Code 
* Open the project folder in VS Code.
* Install the Live Server extension (by Ritwick Dey).
* Click the Go Live button in the bottom status bar or right-click index.html and select Open with Live Server.  

Option 2: Using Python  
If you have Python installed, open your terminal/command prompt in the project folder and run:
```bash
python -m http.server
```
Then, open your browser and go to http://localhost:8000.
